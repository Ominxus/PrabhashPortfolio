Software Engineering Project

Student Planning System

The Student Planning System is a web-based platform designed to help students choose and manage their two-year study plans. The system includes separate student and admin portals, subject management, edit permission requests, audit logs, password resets, and Excel export features.

View Screenshots Ask About Project

Project Overview

Role

Software Developer

Designed and developed the system as an academic software engineering project.

Stack

Java Spring Boot

Built backend logic, REST APIs, authentication, database operations, and admin features.

Database

MySQL

Stored users, subjects, study plans, edit requests, and system records.

Key Features

Student Portal

Students can register, log in, enter personal details, choose subjects, and submit study plans.

Admin Dashboard

Admins can view students, manage submitted plans, handle edit requests, and monitor system activity.

Excel Export

Admin users can export study plan information and student submissions for easier reporting.

Edit Requests

Students can request permission to update submitted plans, and admins can approve or deny them.

Audit Logs

The system tracks important actions for better transparency and maintenance.

Multi-language UI

The interface includes English and Lithuanian language options.
